# BERT Sentiment Analysis

## Overview

This project implements sentiment analysis using DistilBERT, a lightweight version of BERT. The model is fine-tuned on a movie review dataset containing 50,000 reviews and predicts whether a review is positive or negative.

## Features

* Data Loading from CSV
* Data Preprocessing
* Label Encoding
* Tokenization using DistilBERT Tokenizer
* Fine-Tuning DistilBERT
* Model Evaluation
* Sentiment Prediction on New Reviews

## Technologies Used

* Python
* Hugging Face Transformers
* PyTorch
* Datasets
* Scikit-learn
* Google Colab

## Results

The model achieved high classification accuracy on the test dataset and successfully classified unseen reviews.
